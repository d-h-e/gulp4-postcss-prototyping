# gulp4-postcss-prototyping
fast css prototyping with sass, postcss and browser-sync

## install

1. clone git reop

```sh
git clone git@github.com:d-h-e/gulp4-postcss-prototyping.git
```

2. install dev-depandencies

```sh
cd gulp4-postcss-prototyping
npm install
```

3. start gulp

```sh
gulp
```
